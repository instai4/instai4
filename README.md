## 🛠️ Setup Instructions for the 3D Contribution Graph

1. Create a new file in your repository: `.github/workflows/metrics.yml`
2. Paste the following content:

```yaml
name: Metrics
on:
  schedule: [{cron: "0 0 * * *"}]  # daily update
  workflow_dispatch:
  push: {branches: ["main"]}
jobs:
  github-metrics:
    runs-on: ubuntu-latest
    steps:
      - uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          filename: metrics.3d.svg
          base: ""
          plugin_3d_chart: yes
          plugin_3d_chart_title: "3D Contributions"
